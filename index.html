HOW IT WORKS

Activation Code

Seller generates an activation code and gives it to the Buyer.

Activations

Both parties input their expectations, and upon agreement, the buyer makes the payment.

Escrow Payment

Upon agreement the escrow bot will collect the sellers wallet address and prompt the buyer to make payment.

Payment Initiation

Buyer will make payment to the secure escrow bot address and the timer begins.

Release Decision

When the timer is up or if the seller delivers early the buyer is informed and asked If they’re happy for escrow bot to release payment.

Automatic Release

Buyer will make payment to the secure escrow bot address and the timer begins.

Dispute Resolution

If a dispute is opened a standardised procedure is executed.
